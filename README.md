# 台股當沖分析系統 (Day Trading Analyzer)

## 專案簡介

這是一個專為台灣股市設計的當沖股票分析系統，能夠根據成交量、波動度、市場題材等多項指標，自動篩選出具備當日獲利3%潛力的股票標的。

## 功能特色

- **即時市場數據分析**：整合 Yahoo Finance API，獲取台股即時行情
- **多維度評分系統**：綜合評估成交量、波動度、趨勢、題材等指標
- **視覺化報告**：自動生成多種圖表，包括雷達圖、柱狀圖、排名表等
- **風險評估**：提供停損建議、進場策略、風險提示
- **每日更新**：適合每天早上 9:00-9:30 進行當日分析

## 安裝與使用

### 環境需求

- Python 3.11+
- 必要套件：matplotlib, numpy, requests

### 安裝步驟

```bash
# 1. 複製專案
git clone https://github.com/YOUR_USERNAME/day-trading-analyzer.git
cd day-trading-analyzer

# 2. 安裝依賴套件
pip install -r requirements.txt

# 3. 執行股票分析
python src/analyze_stocks.py

# 4. 生成視覺化報告
python src/create_visualization.py
```

## 分析流程

### 1. 數據收集
- 從 Yahoo Finance 獲取台股成交量排行
- 收集即時市場新聞與熱門題材
- 獲取個股技術指標數據

### 2. 篩選標準
- **成交量**：每日成交量 > 2,000 張
- **波動度**：平均波動度 > 3%
- **題材**：具備市場熱門題材
- **趨勢**：5 日趨勢為正

### 3. 評分系統
- 成交量評分：30 分
- 波動度評分：30 分
- 趨勢評分：20 分
- 平均波動度評分：20 分

### 4. 輸出報告
- 推薦前 5 名股票
- 提供進場策略、目標價位、停損建議
- 生成視覺化圖表

## 分析指標說明

| 指標 | 說明 | 重要性 |
|------|------|--------|
| 成交量 | 確保流動性，避免進出困難 | ★★★★★ |
| 波動度 | 提供足夠價差空間 | ★★★★★ |
| 題材 | 吸引短線資金，帶動行情 | ★★★★☆ |
| 5日趨勢 | 判斷中短期動能 | ★★★☆☆ |
| 當日漲幅 | 評估當日強度 | ★★★☆☆ |

## 風險提示

⚠️ **重要聲明**

1. 本系統僅供參考，不構成投資建議
2. 股票投資有風險，請謹慎評估自身風險承受能力
3. 當沖交易需要嚴格的風險控制與停損紀律
4. 建議停損設定在 1.5-2%，目標獲利 3%
5. 避免追高，應在回檔至支撐位時進場
